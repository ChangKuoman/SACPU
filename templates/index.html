SACPU - PC BUILD SIMULATOR

LOGIN REGISTER

Nuestro proyecto es una aplicación web realizada con el micro Framework Flask que permite el simulado de compra de una computadora teniendo en cuenta como pieza base la tarjeta madre y la compatibilidad de esta con los diferentes componentes. La información de la base de datos está basada en direntes páginas web de compra de tecnología especializada en computadoras como LoginStore, y marcas propias como MSI, AMD, Intel, entre otros. click para saber más :)
